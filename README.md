# HireMind AI – Resume Analysis Platform
Problem Solved: Job seekers often struggle to know whether their resume truly matches a job description, where their skill
gaps are, or how to prepare for interviews. HireMind AI addressed this by automating resume analysis, gap identification,
and interview preparation in one platform.
• Built a MERN stack platform (React.js, Node.js, Express.js, MongoDB) that analyzes resumes against job
descriptions, surfaces skill gaps, and generates tailored interview questions.
• Integrated the Gemini API to power AI-driven interview preparation and used Puppeteer to auto-generate down
loadable, ATS-optimized PDF resumes.
• Implemented

## 🚀 Features

- AI-powered resume analysis
- Resume score and feedback
- Authentication
- Responsive user interface
- Secure backend APIs

## 🛠️ Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

### Clone the repository

```bash
git clone https://github.com/shahzaibwork/HireMind-AI-Resume-Analysis-Platform.git
```
### Install dependencies

Frontend:

```bash
cd Frontend
npm install
npm run dev
```

Backend:

```bash
cd Backend
npm install
npm start
```
